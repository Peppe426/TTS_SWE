# Code of Conduct

## Our standard

I want this project to be welcoming, respectful, and constructive for everyone involved.

Participants are expected to:

- be respectful in discussions
- give feedback in good faith
- welcome new contributors
- focus on collaboration and clarity
- respect privacy and consent concerns around voice data

## Unacceptable behavior

This includes:

- harassment or personal attacks
- discrimination or exclusionary behavior
- abusive language
- sharing someone else's personal or voice data without permission
- derailing project discussions in bad faith

## Scope

This Code of Conduct applies to project discussions, issues, pull requests, and any community spaces directly tied to the project.

## Enforcement

Project maintainers may remove content, limit participation, or reject contributions that do not align with this Code of Conduct.

## Reporting

Reporting and moderation details are not defined yet. Until then, maintainers should handle issues conservatively and prioritize community safety.
