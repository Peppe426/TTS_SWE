# Contributing

Thanks for wanting to help build a Swedish community TTS project.

Right now, this project is in the **foundation stage**, so the most valuable contributions are the ones that help us clarify scope, build trust, and create a clean starting point for later engineering work.

## Current contribution areas

We currently welcome help with:

- project documentation
- roadmap and scope feedback
- Swedish prompt writing
- consent and privacy review
- data governance suggestions
- mobile app and backend planning
- open Swedish dataset research

## What we are not asking for yet

At this stage, we are **not** asking contributors to:

- submit production model-training code
- assume a fixed TTS architecture
- upload voice data without a defined consent flow
- mix external datasets into the project without license review

## How to contribute right now

### Documentation

Improve clarity, correct wording, and help structure the project.

### Research

Help identify:

- Swedish speech datasets
- Swedish lexicons and pronunciation resources
- licensing risks
- comparable open TTS projects

### Product and engineering input

Help shape:

- the mobile recording flow
- backend requirements
- data review and moderation workflow
- dataset export format

## Contribution principles

### 1. Be transparent

If you are suggesting a dataset, model, or policy, explain why.

### 2. Respect consent and privacy

Do not share recordings, transcripts, or personal data casually.

### 3. Prefer clarity over speed

This project depends on community trust. A slower, clearer process is better than a rushed one.

### 4. Keep Swedish users in mind

Suggestions should support the goal of building a strong Swedish-first speech foundation.

## Before contributing voice data

Voice contribution workflows are **not live yet**.

When they are introduced, contributors must be clearly informed about:

- what is collected,
- how it is stored,
- how it may be used,
- whether it may be published,
- and whether it may later be used for model training.
