# Roadmap

## Overview

This roadmap matches the current `README.md`.

The numbered phases now start with the **product and data work**. Documentation, governance, and repository setup are still important, but they are treated as foundation work rather than their own numbered phases.

## Foundation work

These items support the roadmap but are not treated as numbered phases:

- project README and core documentation
- contribution guidance and community expectations
- initial scope, consent, privacy, and governance direction
- repository setup for future app, backend, and data work

### Phase 1 - Mobile recording app MVP

Deliverables:

- recording flow
- prompt presentation
- consent screens
- upload integration

### Phase 2 - Backend and data pipeline

Deliverables:

- upload endpoints
- metadata storage
- moderation states
- quality checks
- dataset export flow

### Phase 3 - Open-data research and enrichment

Deliverables:

- shortlist of Swedish-compatible resources
- license review notes
- integration recommendations

### Phase 4 - Future model exploration

Deliverables:

- baseline model options
- evaluation approach
- prototype plan

This phase remains intentionally open.

### Phase 5 - Community operations

Deliverables:

- moderation process
- contribution workflow
- release expectations
- onboarding guidance

## Current milestone

The current milestone spans **Phase 1** and **Phase 2**:

- define the mobile recording flow,
- define consent and recording guidance,
- plan upload and metadata handling,
- and prepare the backend/data pipeline requirements.
