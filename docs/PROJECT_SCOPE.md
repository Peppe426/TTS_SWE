# Project Scope

## Current project goal

The current goal is to create the foundation for a future Swedish TTS effort by building a **community-driven Swedish speech collection platform**.

## First milestone scope

This scope spans:

- **Phase 1 - Mobile recording app MVP**
- **Phase 2 - Backend and data pipeline**

The first milestone focuses on:

- a mobile app for guided Swedish recordings
- a backend for upload and metadata handling
- data storage and review workflows
- consent capture
- dataset curation and export
- documentation and community setup

## First milestone non-goals

The first milestone does **not** include:

- a finished Swedish TTS model
- a production inference API
- final model architecture decisions
- large-scale training infrastructure
- unrestricted public voice uploads without review

## Why this scope

The project needs a clean data and governance foundation before model decisions become useful.

## Data collection direction

The first data collection flows should prefer:

- prompted Swedish reading
- short guided answers
- explicit contributor consent
- high-quality metadata
- reviewable and exportable recordings

## Future scope

Future phases may include:

- using compatible external Swedish datasets
- experimenting with model fine-tuning
- building evaluation workflows
- publishing dataset releases

These are future possibilities, not current commitments.
